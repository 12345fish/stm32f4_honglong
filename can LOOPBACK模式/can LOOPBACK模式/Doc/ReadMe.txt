测试说明：
    本程序为CAN LOOPBACK模式，非正常CAN测试模式。测试大致流程为 串口3接收数据后，用CAN发送，CAN发送后会自收，
    引起CAN 接收中断，CAN接收中断，把接收到的数据再发送出去（调用了串口3发送）.

    测试步骤：第一：串口链接，跳线帽J4,J5插上，J22插到靠近RS232那一侧。
              第二：CAN1链接，跳线帽J2,J3.
              第三：串口调试助手，发送字符。（波特率115200,8个数据位，1个停止位,无奇偶校验，无硬件流）

    编译并下载程序。

测试结果：
    串口调试助手发送字符a,则串口调试助手收到字符a.
