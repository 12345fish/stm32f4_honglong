测试说明：
    第一步：取下跳线帽J12 (注意，红龙407开发板J12这只脚和按键S1共用)
    第二步：编译并下载程序。

测试结果：
    按动S1按键，LED1灯 亮灭交替；
    按动S2按键，LED2灯 亮灭交替；
    按动S3按键，LED3灯 亮灭交替；
    按动S4按键，LED4灯 亮灭交替；
    按动S5按键，LED全部灯 亮灭交替；
